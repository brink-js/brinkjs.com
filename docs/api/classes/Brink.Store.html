    Brink.Store

    Module: data
    Parent Module: Brink

    The store is a glorified cache, with convenience methods to work with your Adapters to update or query your persistence layer as needed. By having a Store, you will need to access your persistence layer much less frequently and you will be able to return records from the store instantly.
